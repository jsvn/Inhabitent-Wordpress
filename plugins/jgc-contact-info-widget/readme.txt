This plugin creates a widget to display your profile/bussiness information.

== Description ==

This plugin creates a widget to display your profile/bussiness information with the following data:

* Company.
* Address (allowed HTML tags).
* Zip Code.
* City.
* Province.
* Phone.
* Mobil.
* Fax.
* 3 socials URL (With automatic social icons for Facebook, Twitter, Google+, LinkedIn, YouTube, WordPress, Instagram, Pinterest, GitHub, Skype, Vimeo).
* Free text that you can put after or before data (allowed HTML tags).

You can set too the following options:

* Text color.
* Social icons's color (same that text or original color).

== Installation ==

For an automatic installation through WordPress:

1. Go to Plugins > Add New
2. Search for 'JGC Contact Info Widget'
3. Click 'Install Now' and activate the plugin
4. Go to Appearance > Widgets

For a manual installation via FTP:

1. Unzip the file 'jgc-contact-info-widget.zip'
2. Upload the 'jgc-contact-info-widget' folder to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Appearance > Widgets

To upload the plugin through WordPress, instead of FTP:

1. Go to Plugins > Add New
2. Click 'Upload Plugin' button
3. Select 'jgc-contact-info-widget.zip' file
4. Click 'Install Now' and activate the plugin
5. Go to Appearance > Widgets
